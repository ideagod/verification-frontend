# verification-frontend

人脸识别demo

### 1.下载

在Git Bash Here下输入：

```
git clone https://github.com/ideagod/verification-frontend.git
```

### 2.导入

下载微信开发者工具并将文件导入，选择“**不使用云开发**”。

### 3.运行

1.在主目录下打开终端，输入

```
npm install
```

2.点击微信开发者工具左上角-工具-构建npm。
